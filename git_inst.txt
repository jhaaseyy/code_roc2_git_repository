GitHUB Instructions:

1. Open terminal and install git
	In LINUX: sudo apt-get install git
2. Configure user profile (local)
	git config --global user.name "Steven"
	git config --global user.email "h1liang.ucsd.edu"
3. Create repo on github.com  (if you are the first one to create it, otherwise go to 4.)
	go to github.com and create repo
4. Create git user profile
	git config --global user.name "Steven"
	git config --global user.email "h1liang@ucsd.edu"
5. cd into the working directory of the git repo
	click "Clone or Download (In green)" and copy "https://github.com/hcl96/subl.git"
	git clone https://github.com/hcl96/subl.git
6. To upload a new file onto the repo
	*add the file in the repo directory
	git add "filename"
	git commit -m "add comment here"
	git push 
7. Check GitHUB for updated file
8. To download the latest version
	git pull
	*check ls for the updated list of files
9. When editing the same file

Workflow
========
*At each start of the day/session, do git pull to update your repo with the latest version
*Routine:
	1. Get a work request
	2. Make a branch
	3. Code feature in *the* branch
	4. Commit and push
	5. Merge review and merge back to MASTERS
